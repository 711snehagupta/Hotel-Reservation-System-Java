# Hotel Reservation System

## Overview

The Hotel Reservation System is a Java-based console application that enables users to manage hotel room bookings efficiently. The system allows customers to view available rooms, make reservations, cancel bookings, simulate payments, and manage booking records using file handling and object-oriented programming concepts.

## Features

* View available hotel rooms
* Room categorization (Standard, Deluxe, Suite)
* Book hotel rooms
* Cancel reservations
* Payment simulation
* View booking details
* File handling for booking records
* Console-based user interface
* Object-Oriented Programming (OOP) implementation

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* File Handling
* Scanner Class

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Hotel-Reservation-System-Java.git
   ```

2. Compile the program:

   ```bash
   javac HotelReservationSystem.java
   ```

3. Run the application:

   ```bash
   java HotelReservationSystem
   ```
